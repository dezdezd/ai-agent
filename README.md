# 🤖 AI Персональный Ассистент

Персональный AI-ассистент с памятью, поиском в интернете и математическими вычислениями.

## ✨ Возможности

- 🔍 Поиск актуальной информации в интернете (Tavily)
- 🧮 Математические вычисления
- 🧠 Память в рамках диалога
- 🛡️ Защита от промпт-инъекций
- 🌍 Поддержка русского и английского языков

## 🛠️ Технологии

- Python 3.11+
- Groq API (LLaMA 3.3)
- Tavily Search API

## 🚀 Установка

1. Клонируй репозиторий
   git clone https://github.com/dezdezd/ai-agent

2. Установи зависимости
   pip install -r requirements.txt

3. Создай файл .env
   GROQ_API_KEY=твой_ключ
   TAVILY_API_KEY=твой_ключ

4. Запусти
   python main.py

## 💬 Пример диалога

   Ты: какая погода в Москве?
   Агент: Сегодня в Москве облачно, 4°C...

   Ты: сколько будет 4500 * 12?
   Агент: 54 000

   Ты: как меня зовут?
   Агент: Вадим, ты представился в начале диалога.
